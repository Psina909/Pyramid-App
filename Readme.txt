========================================================================
                           Gaussian Pyramid
========================================================================
////////////////////////////////////////////////////////////////////////

This application is designed to provide an opportunity to work with Gaussian pyramid of images.

HOW TO USE:
Open any image (.png, .jpg) and select any layer of the Gaussian pyramid. It will be scaled to original resolution.

You can open several images.
You can also select the scaling coefficient.
